# **App Name**: LinguaLive

## Core Features:

- Voice Translation: Real-time voice translation tool, providing near-instantaneous conversion between spoken languages. Whisper is employed as a tool within the voice translation flow to help decide if particular sections of the original voice data needs to be added into the converted version.
- Voice Call Connection: Connect two users for a real-time voice call.
- Automatic Language Detection: Automatically detect the language spoken by each user.
- Translated Voice Playback: Playback the translated voice in a natural-sounding voice.
- Preferred Language Setting: Enable users to select their preferred language before initiating a call.
- Call Encryption: Ensure privacy and data encryption during calls to protect user information.

## Style Guidelines:

- Primary color: Vibrant blue (#29ABE2) to represent communication and clarity.
- Background color: Light blue (#E5F5FB) to maintain a calm and clean interface.
- Accent color: Yellow (#FFDA63) to highlight key interactive elements.
- Body and headline font: 'PT Sans', a humanist sans-serif font that balances a modern look with warmth and personality. This font is suitable for both headlines and body text, ensuring readability and a consistent aesthetic.
- Use minimalist, universally recognizable icons for languages and settings.
- Maintain a clean, intuitive layout optimized for mobile devices.
